# Verify Implementation

1. Identify what was just implemented/changed
2. Find the appropriate verification command (test suite, build, lint, curl, etc.)
3. Run it and capture output
4. Report: PASS with evidence, or FAIL with exact error
5. If no verification command exists, state that explicitly
